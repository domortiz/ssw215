Playlist Generator

Need some more music to listen to? We gotchu.

This program, once you authenticate and authorize this application to access your Spotify account, generates a new playlist for you to listen to.

We used Spotipy to create this, which is a Python library for the Spotify Web API. Our script grabs your top 5 Artists, and then uses Spotify's recommendation algorithm to create a playlist for you!
